# ClearBooks Financial Services Website

A complete, professional website for your bookkeeping and GST services business.

## Files Included
- `index.html` — The full website (single file, no dependencies)
- `vercel.json` — Vercel deployment configuration

---

## How to Deploy on Vercel (Free)

### Option A — Drag & Drop (Easiest, no coding needed)

1. Go to **https://vercel.com** and sign up for a free account
2. From your dashboard, click **"Add New Project"**
3. Choose **"Browse"** or drag & drop the folder containing these files
4. Vercel will auto-detect the configuration
5. Click **"Deploy"**
6. Your site will be live at a URL like: `https://clearbooks-financial.vercel.app`

### Option B — Vercel CLI

```bash
npm install -g vercel
cd /path/to/your/folder
vercel
```

Follow the prompts. Your site goes live in under 60 seconds.

### Option C — GitHub (Recommended for updates)

1. Create a free GitHub account at github.com
2. Create a new repository, upload both files
3. Connect your GitHub to Vercel
4. Every time you update and push files, Vercel auto-redeploys

## Customise Before Deploying

Open `index.html` in any text editor (Notepad, VS Code) and update:

| What to change | Search for this text |
|---|---|
| Your business name | `ClearBooks Financial` |
| Your phone number | `+91 98765 43210` |
| Your email | `hello@clearbooks.financial` |
| Your address | `123 Finance Street` |
| Stats (clients served etc.) | `500+`, `10+` in hero section |

## Custom Domain

After deploying, you can add your own domain (e.g. `www.yourbusiness.com`) free on Vercel:
Settings → Domains → Add your domain → Follow DNS instructions.
